# Syntax Consistency Rules

문서 기준일: 2026-05-20

이 파일은 task `0251-docs-site-canonical-language-ledger` 이후 남아 있는 short bridge stub다.

Canonical syntax consistency policy는 [docs-site Grammar](../../docs-site/src/content/docs/grammar.md)의 `Design Rules`와 [docs-site CLI](../../docs-site/src/content/docs/cli.md)의 `Formatting Convention` 섹션이다.

## Bridge Scope

- Shared notation for `:`, `=`, `{ ... }`, `->`, `=>`, `T?`, `name?: T`, `let`, `let mut`, `literal`, declaration layout, import/open order, delimiter policy, optional-vs-nullable meaning, mutability, and interop modifiers is folded into docs-site Grammar and CLI.
- 이 bridge는 active agent가 예전 경로를 열었을 때 target canonical pages를 찾을 수 있게 하기 위해서만 남아 있다.
