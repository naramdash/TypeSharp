# TypeSharp Docs

This folder owns the canonical public documentation site.

## Contents

- `src/content/docs`: public language reference, guides, feature status, project ledgers, and workflow documentation.
- `astro.config.ts`: Astro Starlight configuration and TypeSharp syntax highlighting integration.
- `package.json` and `package-lock.json`: docs build dependencies.

## Common Commands

```powershell
cd docs
npm run build
```

Long-lived user-facing language and project documentation should be updated here before temporary agent work notes.
