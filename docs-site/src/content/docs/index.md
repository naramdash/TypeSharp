TypeSharp targets generated `.NET Framework 4.8` artifacts while bringing modern static language features, CLI workflows, and VS Code tooling into long-lived .NET Framework projects.

This site is the navigable publishing surface for the repository documentation. The source-of-truth design documents remain in `docs/`, and each page here points to the relevant canonical document or verified example.

## Start Here

- [Core Goal](/goal/) summarizes the language objective and success criteria.
- [Grammar](/grammar/) links the stable grammar surface and coverage matrix.
- [CLI](/cli/) covers `typesharp check`, `build`, `run`, `version`, and `explain`.
- [VS Code And LSP](/vscode-lsp/) describes extension activation and language server features.
- [Examples](/examples/) lists smoke-tested runnable adoption examples.
